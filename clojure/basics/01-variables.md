# Variable

## Simple vars

Syntax will be:

```clj
(def variable-name variable-value)
```

So it will be like:

```clj
(def foo "bar")
```

## Functional varibale - let

Syntax will be:

```clj
(let [variable-name variable-value]
  (so-sth variable-name))
```

So it will be like:

```clj
(let [foo "bar"]
  (println foo))
```
